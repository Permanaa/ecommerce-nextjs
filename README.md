# Demo
[ecommerce-series.vercel.app](https://ecommerce-series.vercel.app/)

# Installation
- Clone project
- `cp .env.local.example .env`
- Run `yarn`
- Run `yarn dev`
- Open `localhost:3000` in browser

# Development
- Run `yarn dev`
- Open `localhost:3000` in browser

# Production Build
- Run `yarn build`
- Run `yarn start`
- Set environment type on .env
- Open `localhost:3000` in browser

# Stack
- Next JS
- Framer Motion
- Next Auth
- Rapid API
